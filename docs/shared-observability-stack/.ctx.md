## CONTEXT.md
"Personal AI Stack" — home network + remote access, local LLMs, cloud AI, RAG, MCP tool use. Key terms: Home MCP (MCP server on NAS/desktop reachable via Tailscale), IdP (Authentik), Public Service (Cloudflare Tunnel, houseoflight.dev), Internal Service (LAN/Tailscale only), Access Group (admin/household/social).

CONVENTIONS.md section 1: dedicated nologin service user per desktop/NAS service, never `preston`/`root`; narrow `/etc/sudoers.d/<svc>` grant if docker access needed, never docker-group membership. Section 2: Mac is dev/edit-only, execution happens on desktop via `ssh desktop-agent`. Section 3: SSH as `agent` user, NOPASSWD sudo both boxes. Section 5: secrets never committed, `.env` git-ignored, only `.env.example` tracked.
